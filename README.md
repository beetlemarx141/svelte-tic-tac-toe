# Svelte Tic Tac Toe

A minimal Tic Tac Toe game built with Svelte 5 and SvelteKit.

## Run locally

```bash
npm install
npm run dev
```

Then open [localhost:5173](http://localhost:5173) in your browser.

## Project structure

- `src/routes/+page.svelte` — the game UI
- `package.json` — project manifest
- `svelte.config.js`, `vite.config.js` — configs

Feel free to customize further or add new features!
